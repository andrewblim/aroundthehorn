Date rollover

By default, when you load the event feed page, the as-of date is set to today. But if you are following games that go on past midnight, local time, you probably do not want the default as-of date to roll for a few more hours. Set an offset below to delay that roll; 4 hours for people in Eastern Time and a correspondingly offset number for other time zones should work.
